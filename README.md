Moodly AI - نظام خبير لتحليل المشاعر

📋 نبذة عن المشروع
مودلي (Moodly AI) هو نظام خبير ذكي وبسيط مصمم لتحليل النصوص العربية وفهم الحالة العاطفية خلفها. يساعد النظام أصحاب المشاريع والشركات على معرفة انطباعات المستخدمين عن خدماتهم بشكل فوري، وتصنيفها إلى (إيجابية، سلبية، أو محايدة).

🚀 المميزات الرئيسية
التحليل الفردي اللحظي: يمكن للمستخدم كتابة أي تعليق وسيقوم النظام بتصنيفه فوراً مع إظهار تعبير (Emoji) مناسب للحالة.
التحليل الجماعي (Bulk Analysis): يحتوي النظام على ميزة تحليل قائمة كاملة من البيانات الجاهزة لإظهار سرعة ودقة المحرك الاستدلالي.
واجهة مستخدم عصرية: تم تصميم الموقع بألوان هادئة (Pastel Colors) وخطوط احترافية لضمان تجربة مستخدم مريحة.
🛠 البناء التقني (Tech Stack)
تم بناء المشروع باستخدام مفاهيم الأنظمة الخبيرة وتقنيات الويب الأساسية:

HTML5 & CSS3: لبناء الهيكل وتصميم الواجهة بشكل عصري وجذاب.
JavaScript: لبرمجة المحرك الاستدلالي (Inference Engine) الذي يقوم بمعالجة النصوص داخل المتصفح.
Python: تم توثيق المنطق البرمجي الأساسي وقاعدة المعرفة في ملف model.py.
📁 ملفات المشروع
index.html: يمثل واجهة المستخدم الرئيسية.
style.css: يحتوي على التنسيقات الجمالية والألوان المتدرجة.
script.js: يمثل محرك التفاعل والتحليل الفوري.
model.py: يحتوي على منطق النظام الخبير وقاعدة البيانات المعرفية.
تم تطوير هذا المشروع كنموذج عملي لتبسيط مفاهيم الذكاء الاصطناعي وهندسة اللغويات.
